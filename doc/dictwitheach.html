NAME
DictWithEach — Run dict with successively for each key.
SYNOPSIS
DictWithEach DictionaryVarName Body
DESCRIPTION
Return Value
None.
Arguments
DictionaryVarNameName of dictionary variable to use.
BodyScript to execute for each dict value.
Error Conditions
ConditionError CodeError Message
Input variable not found in caller. -1 Could not find variable $VarName in caller.
Contents of variable are invalid. -11 Variable $VarName has invalid value $Value.
More Info
This command is useful when you have a dict where each value is itself both a dict and represents a record. Each record has the same fields (see example). It will take each "record" and map each key/value pair to a Tcl variable and then run the script.

Note that this command will create a variable DictWithEach_Key in its caller. This variable can be accessed in the script body, if desired.

EXAMPLES 
% # Data for one employee
% dict set employeeInfo 12345-A forenames "Joe" 
% dict set employeeInfo 12345-A surname "Schmoe" 
% dict set employeeInfo 12345-A phone "555-1234" # Data for another employee 
% dict set employeeInfo 98372-J forenames "Anne" 
% dict set employeeInfo 98372-J surname "Other" 
% dict set employeeInfo 98372-J phone "555-8765" 
% DictWithEach employeeInfo {puts "$forenames $surname -- $phone"}
Joe Schmoe -- 555-1234
Anne Other -- 555-8765
