NAME
SendEmail — Send an email to given address, with given subject and body.
SYNOPSIS
SendEmail ToEmailAddress Subject Body
DESCRIPTION
Return Value
None.x
Arguments
ToEmailAddressAddress to send the email to.x
SubjectText to appear in the subject line.x
BodyText to appear in the message body.x
More Info
xxxx

This command is to be able to quickly and easily send (plaintext) emails. You have to do one-time set up of the configuration with various details (e.g. server, port, etc.) or it will not work. See GetEmailUsingImap4, GetEmailUsingPop3

KEYWORDS
email, network
