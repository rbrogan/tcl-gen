NAME
QuoteIfColumnTypeIsText — Escaped and quote string if it would go in a text column.
SYNOPSIS
QuoteIfColumnTypeIsText TableName ColumnName TargetVariable
DESCRIPTION
Return Value
String ready to be put into the column.
Arguments
TableNameName of table that column is in.
ColumnNameName of column to check.
TargetVariableWhat to quote. Can be either a value or @VarName (will upvar and read/write).
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
More Info
This command will look up the type of the column and then escape the string value then quote it so that it is ready. Otherwise, it will return the same string value that was input.

Note that arguments of the form XxxVariable can take either a normal value or the name of a variable to read/write, prefixed by @. See About Using @ for details.

EXAMPLES 
% QuoteIfColumnTypeIsText {I don''t know}
''I don''''t know''
% set MyString {I don't know}
%  QuoteIfColumnTypeIsText globals desc @MyString
'I don''t know'
% puts $MyString
'I don''t know'
