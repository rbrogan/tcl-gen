NAME
OnFirstIteration — Check whether this the first iteration of the loop.
SYNOPSIS
OnFirstIteration
DESCRIPTION
Return Value
1 On first iteration of the loop.
0 Not on first iteration of the loop.
More Info
This command is designed to make it easier to write readable code. It combines with Foreach, and you can do the equivalent with a regular for loop and counters.
EXAMPLES 
% set MyList {1 2 3}
% Foreach Element $MyList { if {[OnFirstIteration]} {puts "First one"}; puts $Element }
First one
1
2
3
