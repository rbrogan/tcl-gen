NAME
IsMatrix — Check whether list value is a matrix.
SYNOPSIS
IsMatrix ListValue
DESCRIPTION
Return Value
1 Value is a matrix.
0 Value is not a matrix.
More Info
A matrix is a list where every element is a list and they all have the same length. It is like a table but we use the term matrix to make it clear we are not talking about a database table.
