NAME
Coe — Concatenate the args but return empty if any is empty.
SYNOPSIS
Coe ?args ...?
DESCRIPTION
Return Value
Concatenation of all arguments or empty string if any one is empty.
Arguments
argsStrings to concatenate.
More Info
COE = Concatenation Or Empty.

Can be useful for forming optional clauses in a statement (e.g. a WHERE clause).

EXAMPLES 
% set MyDict {}
% set sql "SELECT * FROM my_table[Coe \" WHERE \"  [SqlWhereClause $MyDict]]"
SELECT * FROM my_table
