NAME
LinkTclVariableToRegistryValue — Make link to propagate changes from Tcl var to registry.
SYNOPSIS
LinkTclVariableToRegistryValue VarName RegistryKeyPath RegistryValueName
DESCRIPTION
Return Value
None.
Arguments
VarNameName of the variable to link.
RegistryKeyPathPath for registry key that the value is under.
ValueNameName of the value to link.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
More Info
Does not support types other than sz.
EXAMPLES 
% registry set {HKEY_CURRENT_USER\Software\My} testy 123
% puts [registry get {HKEY_CURRENT_USER\Software\My} testy]
123
% set MyVar 345
% LinkTclVariableToRegistryValue MyVar {HKEY_CURRENT_USER\Software\My} testy
% puts [registry get {HKEY_CURRENT_USER\Software\My} testy]
345
% set MyVar 456
% puts [registry get {HKEY_CURRENT_USER\Software\My} testy]
456
