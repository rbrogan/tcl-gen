NAME
DateIsBefore — Check whether the first date comes before the second.
SYNOPSIS
DateIsBefore FirstDate SecondDate
DESCRIPTION
Return Value
1 First date is before the second.
0 First date is not before the second.
Arguments
FirstDateString in date format.
SecondDateString in date format.
Error Conditions
ConditionError CodeError Message
Contents of FirstDate are invalid. -11 Variable $VarName has invalid value $Value.
Contents of SecondDate are invalid. -11 Variable $VarName has invalid value $Value.
More Info
This command is like less than (<).

For information about how to configure the format, look here.

EXAMPLES 
% DateIsBefore {2000-01-02} {2000-01-03}
1
