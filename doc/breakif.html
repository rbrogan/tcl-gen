NAME
BreakIf — Break if the condition is true.
SYNOPSIS
BreakIf Condition
DESCRIPTION
Return Value
None.
Arguments
ConditionCondition to evaluate
More Info
This is equivalent to if {[expr $Condition]} {break} but is a bit closer to natural language.
EXAMPLES 
% set MyList {1 2 3}
1 2 3
% foreach Element $MyList { BreakIf {$Element eq "3"}; puts $Element }
1
2
