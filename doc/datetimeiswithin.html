NAME
DatetimeIsWithin — Check whether target datetime is within given period.
SYNOPSIS
DatetimeIsWithin TargetDatetime PeriodStartDatetime PeriodTimeLength
DESCRIPTION
Return Value
1 Datetime is within the period.
0 Datetime is not within the period.
Arguments
TargetDatetimeDatetime to check.
PeriodStartDatetimeDatetime for start of period.
PeriodTimeLengthHow long period lasts.
