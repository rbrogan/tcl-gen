Below is a listing of all commands and a brief description for each.

Dict2RegistryTree

Write to registry from dict.

LinkTclVariableToRegistryValue

Make link to propagate changes from Tcl var to registry.

RegistryExists

Check whether a registry key or value exists.

RegistryForEach

Iterate over names, data, and types in a way like dict for.

RegistryPrint

Print the contents of a registry key.

RegistryTree2Dict

Make Tcl dict from Windows registry tree.

RegistryWith

Run script with Tcl variables mapped to registry values.

RegistryWithEach

Execute script with registry values mapped to Tcl variables.

UnlinkTclVariableFromRegistryValue

Stop propagating changes set up by previous link.
