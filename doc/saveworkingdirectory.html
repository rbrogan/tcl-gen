NAME
SaveWorkingDirectory — Save the current working directory, so can later restore.
SYNOPSIS
SaveWorkingDirectory
DESCRIPTION
Return Value
Name of current working directory.
More Info
Calling RestoreWorkingDirectory will bring you back.
EXAMPLES 
% pwd
/home/username
% SaveWorkingDirectory
/home/username
% cd temp
% pwd
/home/username/temp
% RestoreWorkingDirectory
/home/username
% pwd
/home/username
SEE ALSO
