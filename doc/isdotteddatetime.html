NAME
IsDottedDatetime — Check whether given string is of the form YYYY.MM.DD.hhmm.
SYNOPSIS
IsDottedDatetime StringValue
DESCRIPTION
Return Value
1 Given string is of the form YYYY.MM.DD.hhmm.
0 Given string is not of the form YYYY.MM.DD.hhmm.
Arguments
StringValueString to check if of the form YYYY.MM.DD.hhmm.
EXAMPLES 
% IsDottedDateTime 2000.01.02.0304
SEE ALSO
IsHhmmss
KEYWORDS
datetime
