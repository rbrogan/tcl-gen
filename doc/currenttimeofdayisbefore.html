NAME
CurrentTimeOfDayIsBefore — Check whether current time-of-day is before target.
SYNOPSIS
CurrentTimeOfDayIsBefore TargetTimeOfDay
DESCRIPTION
Return Value
1 Current time-of-day is before target time.
0 Current time-of-day is at or after target time.
Arguments
TargetTimeOfDayTime of day that the current time must come before in order for this to return true.
More Info
This is a convenience command which combines TimeOfDayIsBefore and CurrentTimeOfDay.

The time-of-day should be formatted according to the configuration (see here for details or SetTimeOfDayFormat). The configuration defaults to %H:%M:%S, which means 12 o'clock is 12:00:00. (And if you are happy with that, you do not have to do anything to make it work, only if you want something different.)

This is intended for use, for instance, with calendar / appointment / reminder apps.


For information regarding exceptions / errors, see here.
EXAMPLES 
% # Suppose it is 12:00 PM
% CurrentTimeOfDayIsBefore 12:01:00
1
% CurrentTimeOfDayIsBefore 11:59:00
0
