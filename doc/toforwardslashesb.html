NAME
ToForwardSlashesB — Take string, switch single/double backslashes to forward.
SYNOPSIS
ToForwardSlashesB PathVarName
DESCRIPTION
Return Value
A string where the single/double backslashes are now forward slashes.
Arguments
PathVarNameVariable holding a path to convert.
EXAMPLES 
% set MyPathVar {c:\path\to\somefile.txt}
c:\path\to\somefile.txt
% ToForwardSlashes MyPathVar
c:/path/to/somefile.txt
% set MyPathVar {c:\another\path}
c:\another\path
% ToFowardSlashes MyPathVar 
c:/another/path/
