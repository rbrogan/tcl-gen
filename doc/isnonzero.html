NAME
IsNonZero — Check whether value is non-zero.
SYNOPSIS
IsNonZero Value
DESCRIPTION
Return Value
1 Value is non-zero.
0 Value is zero.
Arguments
ValueValue to check.
Error Conditions
ConditionError CodeError Message
Input value is non-numeric. -2 Got input value $Value. Expected numeric value.
EXAMPLES 
% IsNonZero 1
1
% IsNonZero 0
0
SEE ALSO
IsNonNegative, IsNonPositive, IsZero
KEYWORDS
check, math, validation, verification
