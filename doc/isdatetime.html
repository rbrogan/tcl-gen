NAME
IsDateTime — Tell whether the string is of the form YYYY-MM-DD hh:mm:ss.
SYNOPSIS
IsDateTime String
DESCRIPTION
Return Value
1 If string has form YYYY-MM-DD hh:mm:ss.
0 String has not form YYYY-MM-DD hh:mm:ss.
Arguments
StringString to check.
More Info
Does not yet handle other forms of datetimes.
EXAMPLES
% IsDateTime {2000-07-08 12:34:56}
1
% IsDateTime {2000-07-08}
0
SEE ALSO
IsDate
KEYWORDS
check, date, string, time
