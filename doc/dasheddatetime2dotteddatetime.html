NAME
DashedDatetime2DottedDatetime — Convert YYYY-MM-DD hh:mm:ss to YYYY.MM.DD.hhmm.
SYNOPSIS
DashedDatetime2DottedDatetime StringVariable
DESCRIPTION
Return Value
A string of the form YYYY.MM.DD.hhmm.
Arguments
StringVariableString of the form YYYY-MM-DD hh:mm:ss.
Error Conditions
ConditionError CodeError Message
StringVariable contains invalid value. -11 Variable $VarName has invalid value $Value.
StringVariable is empty. -12 Variable $VarName has empty value.
EXAMPLES 
% DashedDateTime2DottedDateTime {2000-07-08 12:34:56}
2000.07.08.1234
