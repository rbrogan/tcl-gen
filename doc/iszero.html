NAME
IsZero — Check whether value is zero or not.
SYNOPSIS
IsZero Value
DESCRIPTION
Return Value
1 Value is zero.
0 Value is not zero.
Arguments
ValueValue to check.

For information regarding exceptions / errors, see here.
EXAMPLES 
% IsZero 0
1
% IsZero 1
0
SEE ALSO
IsNonZero
