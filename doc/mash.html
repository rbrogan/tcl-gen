NAME
Mash — Take string value, make into all lowercase and no spaces.
SYNOPSIS
Mash StringValue
DESCRIPTION
Return Value
String like the original except all in lowercase and no spaces.
Arguments
StringValueString value to convert into mashed version.
EXAMPLES 
% set DisplayName "Two Words"
Two Words
% set FileName "[Mash $DisplayName].txt"
twowords.txt
SEE ALSO
string
KEYWORDS
string, transform
