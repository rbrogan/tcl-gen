NAME
DatetimePlusDays — Add some days to the given datetime and get the result.
SYNOPSIS
DatetimePlusDays DatetimeVariable NumDays
DESCRIPTION
Return Value
A string containing the datetime that is N days after the current datetime.
Arguments
DatetimeVariableDatetime to add days to.
NumDaysNumber of days to add.
Error Conditions
ConditionError CodeError Message
Contents of DateTimeVariable are invalid. -11 Variable $VarName has invalid value $Value.
Contents of NumDays are invalid. -11 Variable $VarName has invalid value $Value.
EXAMPLES 
% DateTimePlusDays {2000-01-02 03:04:05} 1
2000-01-01 03:04:05
