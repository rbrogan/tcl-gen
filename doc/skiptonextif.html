NAME
SkipToNextIf — Skip to next loop iteration (continue) of condition is true.
SYNOPSIS
SkipToNextIf Condition
DESCRIPTION
Return Value
None.
Arguments
ConditionCondition to evaluate if true.
More Info
This is equivalent to if {[expr $Condition]} {continue} but is closer to natural language.

The command is not called ContinueIf because we are trying to follow a general policy of making command names match natural language usage. "Continue if ..." sounds as if the command is going to continue processing if the condition is true, but will halt if the condition is false.

EXAMPLES 
% set MyList {1 2 3}
1 2 3
% foreach Element $MyList { SkipToNextIf {$Element eq "1"}; puts $Element }
2
3
