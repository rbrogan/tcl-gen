NAME
ToDoubleBackslashesB — Take string, switch single slashes to double backslashes.
SYNOPSIS
ToDoubleBackslashesB PathVarName
DESCRIPTION
Return Value
New contents of variable: a path where all the slashes are double backslashes.
Arguments
PathVarNameVariable holding a path to convert.
More Info
Note this will convert not only forward slashes to double backslashes but it will also convert single backslashes to double backslashes.
EXAMPLES 
set MyPathVar c:/path/one
c:/path/one
% ToDoubleBackslashesB MyPathVar
c:\path\one
% set MyPathVar {c:\path\to\file.txt}
c:\path\to\file.txt
% ToDoubleBackslashesB MyPathVar
c:\path\to\file.txt
