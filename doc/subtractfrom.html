NAME
SubtractFrom — Take variable and subtract amount from it.
SYNOPSIS
SubtractFrom VarName Value
DESCRIPTION
Return Value
Value of the variable after subtraction.
Arguments
VarNameName of variable to subtract from.
ValueHow much to subtract.

For information regarding exceptions / errors, see here.
EXAMPLES 
% set SomeNumber 111
111
% SubtractFrom SomeNumber 11
100
