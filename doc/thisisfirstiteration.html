NAME
ThisIsFirstIteration — Check whether this the first iteration of the loop.
SYNOPSIS
ThisIsFirstIteration
DESCRIPTION
Return Value
1 On first iteration of the loop.
0 Not on first iteration of the loop.
More Info
This command is designed to make it easier to write readable code. You must use ForEach for this command to work.
EXAMPLES 
% set MyList {1 2 3}
% ForEach Element $MyList { if {[ThisIsFirstIteration]} {puts "First one"}; puts $Element }
First one
1
2
3
