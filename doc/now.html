NAME
Now — Get the current datetime as formatted string.
SYNOPSIS
Now
DESCRIPTION
Return Value
Current datetime as formatted string.
More Info
For information about how to configure the format, look here.
EXAMPLES 
% # Suppose that it is 2000-01-02 03:04:05
% Now
2000-01-02 03:04:05
