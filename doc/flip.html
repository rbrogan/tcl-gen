NAME
Flip — Change value to zero if it is one and vice versa.
SYNOPSIS
Flip Value
DESCRIPTION
Return Value
1 If input was zero.
0 If input was one.
Arguments
ValueValue to flip.
Error Conditions
ConditionError CodeError Message
Input value is non-numeric. -2 Got input value $Value. Expected numeric value.
Input value is out-of-range. -8 Input value $Value is out-of-range.
More Info
Note that the variable must already have a zero or one to begin with.
EXAMPLES 
% Flip 0
1
% Flip 1
0
SEE ALSO
Decr, incr
KEYWORDS
conversion, math, transform
