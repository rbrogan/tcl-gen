NAME
UnlinkTclVariableFromRegistryValue — Stop propagating changes set up by previous link.
SYNOPSIS
UnlinkTclVariableFromRegistryValue VarName RegistryKeyPath ?RegistryValueName?
DESCRIPTION
Return Value
None.
Arguments
VarNameName of variable to unlink.
RegistryKeyPathPath to registry key that the value is under.
RegistryValueNameName of registry value to unlink.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
EXAMPLES 
% set MyVar 123
% puts [llength [trace info variable MyVar]]
0
% LinkTclVariableToRegistryValue MyVar {HKEY_CURRENT_USER\Software\My\GenTest} testy
% puts [llength [trace info variable MyVar]]
1
% UnlinkTclVariableFromRegistryValue MyVar {HKEY_CURRENT_USER\Software\My\GenTest} testy
% puts [llength [trace info variable MyVar]]
0
