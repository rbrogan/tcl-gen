NAME
UnlinkTclVariableFromRegistryValue — Stop propagating changes set up by previous link.
SYNOPSIS
UnlinkTclVariableFromRegistryValue VarName RegistryKeyPath ?RegistryValueName?
DESCRIPTION
Return Value
None.
Arguments
VarNameName of variable to unlink.
RegistryKeyPathPath to registry key that the value is under.
RegistryValueNameName of registry value to unlink.
More Info
Use the exact same arguments as you did with a previous call to LinkTclVariableToRegistryValue.

Note that currently only global variables are supported.


For information regarding exceptions / errors, see here.
EXAMPLES 
% registry set {HKEY_CURRENT_USER\Software\GenTest}
% set ::MyVar 123
% puts [llength [trace info variable ::MyVar]]
0
% LinkTclVariableToRegistryValue ::MyVar {HKEY_CURRENT_USER\Software\GenTest} testy
% puts [llength [trace info variable ::MyVar]]
1
% UnlinkTclVariableFromRegistryValue ::MyVar {HKEY_CURRENT_USER\Software\GenTest} testy
% puts [llength [trace info variable ::MyVar]]
0
% registry delete {HKEY_CURRENT_USER\Software\GenTest}
