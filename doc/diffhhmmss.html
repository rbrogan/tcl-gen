NAME
DiffHhmmss — Do subtraction of time quantities in hh:mm:ss form.
SYNOPSIS
DiffHhmmss MinuendVariable Subtrahend
DESCRIPTION
Return Value
A string of the form hh:mm:ss that is the time left after subtracting.
Arguments
MinuendVariableWhat gets subtracted from.
SubtrahendHow much to subtract.
Error Conditions
ConditionError CodeError Message
Contents of MinuendVariable are invalid. -11 Variable $VarName has invalid value $Value.
Contents of Subtrahend are invalid. -11 Variable $VarName has invalid value $Value.
Input value is empty. -12 Variable $VarName has empty value.
More Info
Note that the strings represent quantities of time, how long. The strings do not represent time points, times during the day.
EXAMPLES

Here we subtract 30 minutes from 90 minutes and get back an hour.

% DiffHhmmss 01:30:00 00:30:00
01:00:00
