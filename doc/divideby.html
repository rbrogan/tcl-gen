NAME
DivideBy — Make variable become quotient after division.
SYNOPSIS
DivideBy VarName Value
DESCRIPTION
Return Value
New value of variable after division.
Arguments
VarNameName of variable to divide.
ValueHow much to divide by.
Error Conditions
ConditionError CodeError Message
Input variable not found in caller. -1 Could not find variable $VarName in caller.
Input value is non-numeric. -2 Got input value $Value. Expected numeric value.
Got empty string instead of variable name. -3 Variable name is missing. Got empty string.
More Info
DivideBy x 2 is equivalent to x /= 2 in other languages.
EXAMPLES 
% set MyVar 6
6
% DivideBy MyVar 3
2
% puts $MyVar
2
