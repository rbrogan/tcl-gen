NAME
PrintVar — Print out both the variable name and its value.
SYNOPSIS
PrintVar VarName
DESCRIPTION
Return Value
None.
Arguments
VarNameName of variable to print.
Error Conditions
ConditionError CodeError Message
Input variable not found in caller. -1 Could not find variable $VarName in caller.
Got empty string instead of variable name. -3 Variable name is missing. Got empty string.
More Info
A bit more convenient than having to type puts "MyVar is $MyVar" each time.
EXAMPLES 
% set x 123
% PrintVar x
x is 123
