NAME
Seconds2Hhmmss — Convert a number of seconds to a string with form hh:mm:ss.
SYNOPSIS
Seconds2Hhmmss StringVariable
DESCRIPTION
Return Value
A string of the form hh:mm:ss.
Arguments
StringVariableQuantity of seconds to convert.
Error Conditions
ConditionError CodeError Message
Contents of StringVariable are invalid. -11 Variable $VarName has invalid value $Value.
StringVariable is empty. -12 Variable $VarName has empty value.
EXAMPLES 
% Seconds2Hhmmss 60
00:01:00
% Seconds2Hhmmss 3600
01:00:00
SEE ALSO
Hhmmss2Seconds, Hhmmss2Seconds
KEYWORDS
conversion, datetime, string, time
