NAME
ThisIsLastIteration — Check whether this the last iteration of the loop.
SYNOPSIS
ThisIsLastIteration
DESCRIPTION
Return Value
1 On last iteration of the loop.
0 Not on last iteration of the loop.
More Info
This command is designed to make it easier to write readable code. You must use ForEach for this command to work.
EXAMPLES 
% set MyList {1 2 3}
% ForEach Element $MyList { if {[ThisIsLastIteration]} {puts "Last one"}; puts $Element }
1
2
Last one
3
