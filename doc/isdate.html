NAME
IsDate — Tell whether the string is of the form YYYY-MM-DD.
SYNOPSIS
IsDate StringValue
DESCRIPTION
Return Value
1 String has form YYYY-MM-DD.
0 String has not form YYYY-MM-DD.
Arguments
StringValueString to check.
More Info
Does not yet handle other forms of dates.
EXAMPLES 
% IsDate 2000-07-08
1
% IsDate 123
0
