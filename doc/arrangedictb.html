NAME
ArrangeDictB — Change dict variable to put keys in given arrangement.
SYNOPSIS
ArrangeDictB DictName Arrangement
DESCRIPTION
Return Value
New contents of the dict.
Arguments
DictNameVariable name of dict that will get its contents rearranged.
ArrangementList with the keys in the order they should be after done rearranging.
Error Conditions
ConditionError CodeError Message
Input variable not found in caller. -1 Could not find variable $VarName.
Got empty string instead of variable name. -3 Dictionary name is missing. Got empty string.
More Info
Elements of the arrangement that are not found in the input dict are ignored. Note that unlike ArrangeDict, ArrangeDictB takes in the name of a dict variable and modifies its contents.
EXAMPLES

The following code will:

  1. Create a dictionary.
  2. Print it out to show its originally order.
  3. Rearrange it with a call to ArrangeDict
  4. Print it out to show its new order.

Notice how it is rearranged into reverse order.

% set MyDict [dict create one uno two dos three tres]
one uno two dos three tres
% dict for {Key Value} $MyDict { puts "$Key = $Value" }
one = uno
two = dos
three = tres
% ArrangeDictB $MyDict {three two one}
three tres two dos one uno
% dict for {Key Value} $MyDict { puts "$Key = $Value" }
three = tres
two = dos
one = uno
