NAME
DatetimeRangeAround — Get start/end datetimes for period centered at a datetime.
SYNOPSIS
DatetimeRangeAround Datetime TimeAmount
DESCRIPTION
Return Value
A list with two elements: start and end datetime for the period.
Arguments
DatetimeCenter of the range.
TimeAmountHow far to start/end of range on either side of center.
