NAME
SumHhmmss — Sum up a list of times.
SYNOPSIS
SumHhmmss ListValue
DESCRIPTION
Return Value
A string of the form hh:mm:ss that is the sum of the input.
Arguments
ListValueA list where each string is of the form hh:mm:ss.
More Info
Note that minutes and seconds will end up with a value no greater than 60. For example, SumHhmmss {00:30:30 00:30:30} will return 01:01:00, not 00:60:60.
EXAMPLES 
% SumHhmmss [list 01:00:00 00:01:00 00:00:01]
01:01:01
