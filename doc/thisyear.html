NAME
ThisYear — Get the current year.
SYNOPSIS
ThisYear ?Format?
DESCRIPTION
Return Value
The current year. See More Info for format info.
Arguments
FormatOptional. Defaults to return a four digit year. See More Info for details.
More Info
Some relevant format options:
%y: Two digits for the year number. 00 for 2000.
%Y: Four digits for the year number. 2000 for 2000.

Check the documentation of the clock command for more information on the alternate formats.


For information regarding exceptions / errors, see here.
EXAMPLES 
% # Suppose the current year is 2000
% ThisYear
2000
% ThisYear %y
00
