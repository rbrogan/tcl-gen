NAME
First — Get first element of the list.
SYNOPSIS
FirstOf List
DESCRIPTION
Return Value
First element of the list.
Arguments
ListValueList to get the first element of.
ListList value from which to get the first.
More Info
This command is equivalent to lindex $ListValue 0.
EXAMPLES
% set MyList [list one two three]
one two three
% First $MyList
one
SEE ALSO
LastOf
