NAME
OnLastIteration — Check whether on last iteration of the loop.
SYNOPSIS
OnLastIteration
DESCRIPTION
Return Value
1 On last iteration of the loop.
0 Not on last iteration of the loop.
More Info
This command is designed to make it easier to write readable code. It combines with Foreach, and you can do the equivalent with a regular for loop and counters.
EXAMPLES 
% set MyList {1 2 3}
% Foreach Element $MyList { if {[OnLastIteration]} {puts "Last one"}; puts $Element }
1
2
Last one
3
