Control Constructs - v1.3.0

You may use the nav bar to the left to select pages.

Below is a listing of all commands and a brief description for each.

BreakIf

Break if the condition is true.

ForEach

Use replacement foreach to access special commands.

SkipToNextIf

Skip to next loop iteration (continue) of condition is true.

Ter

Return either first or second value.

ThisIsFirstIteration

Check whether this the first iteration of the loop.

ThisIsLastIteration

Check whether this the last iteration of the loop.
