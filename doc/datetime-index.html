datetime - v1.2.1

You may use the nav bar to the left to select pages.

Below is a listing of all commands and a brief description for each.

DashedDatetime2DottedDatetime

Convert YYYY-MM-DD hh:mm:ss to YYYY.MM.DD.hhmm.

DateIsAfter

Check if second date comes after the first date.

DateIsBefore

Check whether the second date comes before the first.

DateIsBetween

Check whether the first date comes between second and third.

DateMinusDays

Get the date some number of days before the given one.

DatePlusDays

Add some number of days to a date of the form YYYY-MM-DD.

DatetimeIsAfter

Check whether the first date is after the second date.

DatetimeIsBefore

Check whether the first datetime is before the second.

DatetimeIsBetween

Check whether the given datetime is between others.

DatetimeIsWithin

Check whether target datetime is within given period.

DatetimeMinus

Find what is the datetime earlier by some amount.

DatetimeMinusDays

Subtract some days from given datetime.

DatetimePlus

Find datetime comes sometime after given one.

DatetimePlusDays

Add some days to the given datetime and get the result.

DatetimeRangeAround

Get start/end datetimes for period centered at a datetime.

DayAfter

Find the date just after the given one.

DayBefore

Find the date just before the given one.

DiffHhmmss

Do subtraction of time quantities in hh:mm:ss form.

DottedDatetime2DashedDatetime

Convert YYYY.MM.DD.hhmm -> YYYY-MM-DD hh:mm:00.

FindNearestPrecedingRecurrence

Find the nearest recurrence that comes before target.

FindNearestSucceedingRecurrence

Find the nearest recurrence datetime after given one.

Hhmmss2Seconds

Convert hh:mm:ss string into integer number of seconds.

HhmmssIsAfter

Check whether the given time-of-day is after the second.

HhmmssIsBefore

Check whether the first time-of-day is before the second.

HhmmssIsBetween

Check whether given time-of-day is between the others.

IsHhmmss

Check whether given string is of the form hh:mm:ss.

MultiplyHhmmss

Multiply a time quantity in hh:mm:ss form by some factor.

Now

Get the current datetime in YYYY-MM-DD hh:mm:ss form.

Seconds2Hhmmss

Convert a number of seconds to a string with form hh:mm:ss.

Seconds2TimeQuantity

Convert a number of seconds into a formatted string.

SumHhmmss

Sum up a list of times.

TimeElapsedBetween

Find out how much time elapsed between two points in time.

TimeQuantity2Seconds

Scan string representing time quantity and return seconds.

tod

Get the current time-of-day in hh:mm:ss form.

today

Get the current date in YYYY-MM-DD form.

tomorrow

Get tomorrow date in the form YYYY-MM-DD.

yesterday

Get yesterday date in the form YYYY-MM-DD.
