NAME
Foreach — Use replacement foreach to access special commands.
SYNOPSIS
Foreach ?args ...?
DESCRIPTION
Return Value
None.
Arguments
argsSame as original foreach. See documentation for original foreach for details.
More Info
The purpose of this command is to write foreach as a proc and make available information that can then be used to create internal control structures which can use uplevel (e.g. IfOnFirst, IfOnLast).
EXAMPLES 
% set MyList {1 2 3}
% Foreach Element $MyList { puts $Element  }
1
2
3
SEE ALSO
IfOnFirst, IfOnLast, foreach
KEYWORDS
control, loop
