NAME
ForEach — Use replacement foreach to access special commands.
SYNOPSIS
ForEach ?args ...?
DESCRIPTION
Return Value
None.
Arguments
argsSame as original foreach. See documentation for original foreach for details.
More Info
The purpose of this command is to write foreach as a proc and make available information that can then be used to create internal control structures which can use with upvar/uplevel (e.g. ThisIsFirstIteration, ThisIsLastIteration).

The command will create two variables within its caller which other commands can expect to find: (1) ForEachMaxIterations is the number of times the loop is expected to execute, based on the size(s) of the list(s) and (2) ForEachCurrentIteration is the number of the current iteration. See the source code of ThisIsLastIteration for an example usage.

Note that, if you like, you can replace the original foreach like this:

rename foreach oldforeach
rename ForEach foreach

EXAMPLES 
% set MyList {1 2 3}
% ForEach Element $MyList { puts $Element  }
1
2
3

control, loop
