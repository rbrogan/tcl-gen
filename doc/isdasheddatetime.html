NAME
IsDashedDatetime — Check whether string is of the form YYYY-MM-DD hh:mm:ss.
SYNOPSIS
IsDashedDatetime StringValue
DESCRIPTION
Return Value
1 Given string is dashed datetime.
0 Given string is not dashed datetime.
Arguments
StringValueString to check whether it is a dashed date time.
EXAMPLES 
% IsDashedDateTime {2000-01-02 03:04:05}
1
SEE ALSO
IsHhmmss
KEYWORDS
datetime
