NAME
Seconds2TimeQuantity — Convert a number of seconds into a formatted string.
SYNOPSIS
Seconds2TimeQuantity SecondsVariable ?Format?
DESCRIPTION
Return Value
Formatted string representing some quantity of time.
Arguments
SecondsVariableNumber of seconds that time quantity is to represent.
FormatFormat string that says how want the time quantity returned to be formatted.
