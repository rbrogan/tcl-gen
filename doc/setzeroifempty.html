NAME
SetZeroIfEmpty — Take variable and make it zero if it is empty.
SYNOPSIS
SetZeroIfEmpty VarName
DESCRIPTION
Return Value
The value that VarName now has.
Arguments
VarNameName of variable to possibly set to zero.
Error Conditions
ConditionError CodeError Message
Got empty string instead of variable name. -3 Variable name is missing. Got empty string.
EXAMPLES 
% set Nothing ""
% SetZeroIfEmpty Nothing
0
% puts $Nothing
0
% set Something "something"
something
% SetZeroIfEmpty Something
% puts $Something
something
