NAME
DayAfter — Find the date just after the given one.
SYNOPSIS
DayAfter DateVariable
DESCRIPTION
Return Value
Date that is one day after the given one.
Arguments
DateVariableTBD
Error Conditions
ConditionError CodeError Message
Contents of DateVariable are invalid. -11 Variable $VarName has invalid value $Value.
EXAMPLES 
% DayAfter {2000-01-02}
2000-01-03

DatePlusDays, DayBefore
