NAME
IsNegative — Tell whether value is negative or not.
SYNOPSIS
IsNegative Value
DESCRIPTION
Return Value
1 The value is negative.
0 The value is not negative.
Arguments
ValueValue to check.
Error Conditions
ConditionError CodeError Message
Input value is non-numeric. -2 Got input value $Value. Expected numeric value.
More Info
If a non-numeric value is put in then an error will be generated.
EXAMPLES 
% IsNegative -1
1
% IsNegative 0
0
SEE ALSO
IsNonPositive, IsPositive
