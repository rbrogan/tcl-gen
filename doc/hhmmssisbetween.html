NAME
HhmmssIsBetween — Check whether given time-of-day is between the others.
SYNOPSIS
HhmmssIsBetween FirstHhmmss SecondHhmmss ThirdHhmmss
DESCRIPTION
Return Value
1 First time-of-day is between the others.
0 First time-of-day is not between the others.
Arguments
FirstHhmmssString of the form hh:mm:ss.
SecondHhmmssString of the form hh:mm:ss.
ThirdHhmmssString of the form hh:mm:ss.
Error Conditions
ConditionError CodeError Message
Contents of FirstHhmmss are invalid -11 Variable $VarName has invalid value $Value.
Contents of SecondHhmmss are invalid. -11 Variable $VarName has invalid value $Value.
Contents of ThirdHhmmss are invalid. -11 Variable $VarName has invalid value $Value.
EXAMPLES 
% HhmmssIsBetween 01:02:03 01:02:02 01:02:04
1
SEE ALSO
HhmmssIsAfter, HhmmssIsBefore
KEYWORDS
datetime
