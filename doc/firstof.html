NAME
FirstOf — Get first element of the list.
SYNOPSIS
FirstOf ListValue
DESCRIPTION
Return Value
First element of the list.
Arguments
ListValueList to get the first element of.
More Info
This command is equivalent to lindex $ListValue 0.

For information regarding exceptions / errors, see here.
EXAMPLES 
% set MyList [list one two three]
one two three
% FirstOf $MyList
one
SEE ALSO
LastOf, lindex
KEYWORDS
extraction, list
