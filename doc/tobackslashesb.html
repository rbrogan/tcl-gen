NAME
ToBackslashesB — Change string variable and make it have only backslashes.
SYNOPSIS
ToBackslashesB PathVarName
DESCRIPTION
Return Value
The new value of the modified variable.
Arguments
PathVarNameName of variable to modify.
Error Conditions
ConditionError CodeError Message
Input variable not found in caller. -1 Could not find variable $VarName.
Passed empty string as first argument. -3 Variable name is missing. Got empty string.
More Info
Changes the string so that all forward slashes and double backslashes are changed to single backslashes. Note that unlike ToBackslashes, this takes in a variable (rather than a value) and modifies it.
EXAMPLES 
% set MyPath c:/some/path/to/file.txt
% ToBackslashesB MyPath
c:\some\path\to\file.txt
