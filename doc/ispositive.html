NAME
IsPositive — Tell whether value is positive or not.
SYNOPSIS
IsPositive Value
DESCRIPTION
Return Value
1 The value is positive.
0 The value is non-positive.
Arguments
ValueValue to check.
Error Conditions
ConditionError CodeError Message
Input value is non-numeric. -2 Got input value $Value. Expected numeric value.
EXAMPLES 
% IsPositive 0
0
% IsPositive 1.1
1
SEE ALSO
IsNegative, IsNonPositive, IsNonPositive, IsZero
