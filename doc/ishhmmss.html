NAME
IsHhmmss — Check whether given string is of the form hh:mm:ss.
SYNOPSIS
IsHhmmss StringValue
DESCRIPTION
Return Value
1 String is of form hh:mm:ss.
0 String is not of form hh:mm:ss.
Arguments
StringValueString to check if it is of the form hh:mm:ss.
EXAMPLES 
% IsHhmmss 01:02:03
1
