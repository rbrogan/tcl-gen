NAME
MashB — Change variable contents to all lowercase and no spaces.
SYNOPSIS
MashB StringVarName
DESCRIPTION
Return Value
The new value the input variable now holds.
Arguments
StringVarNameName of variable to change to be mashed.
Error Conditions
ConditionError CodeError Message
Input variable not found in caller. -1 Could not find variable $VarName.
Got empty string instead of variable name. -3 Variable name is missing. Got empty string.
More Info
Note that unlike Mash, MashB takes a variable name as input and then modifies the contents of that variable.
EXAMPLES 
% set MyVar "Two Words"
Two Words
% MashB MyVar
twowords
% puts $MyVar
twowords
