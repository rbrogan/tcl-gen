Wishlist

Do you want to know what else is there on the horizon (besides the roadmap)? You can check the following for features we will be trying to add in the future. Some of these may come as early as the release next month.

  • Strictness/Permissiveness Policies, Warnings — You can decide what inputs will be silently accepted vs. generate warnings or errors.
  • DetectCase — Tell what case (e.g. PascalCase, snake_case) the given string is in. This will also simplify the ChangeCasing command.
  • Auto Reload — A terminal mode which checks whether files in a package have been modified, and if so, it will auto-reload the package.
  • Loop Constructs — support within foreach loop for clauses such as IfFirst and IfLast which have their scripts evaluated when at first/last elements of list.
  • Support for flags through TEPAM.
  • The various miscellaneous string, list, and file commands that did not make it into the latest release.
