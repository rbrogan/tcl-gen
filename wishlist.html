Wishlist

Do you want to know what else is there on the horizon (besides the roadmap)? You can check the following for features we will be trying to add in the future.

  • Name Transforms — Transforms between snake_case and PascalCase, etc.
  • MultiSet — Set multiple variables at once from a list (i.e. reverse of lassign).
  • Auto Reload — A terminal mode which checks whether files in a package have been modified, and if so, auto-reload the package.
  • Loop Constructs — support within foreach loop for clauses such as IfFirst and IfLast which have bodies evaluated when at first/last elements of list.
  • Upvar Equivalents — for list elements, including a foreach equivalent to dict with that works list elements.
  • Elegant support for command variations which eliminate the need for code such as set MyVar [SomeCommand $MyVar].
  • Support for flags through TEPAM.
