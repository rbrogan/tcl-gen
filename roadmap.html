Gen
Roadmap
1.5
Undetermined

These next months will be a transition period for the Gen project. Originally, the project came from writing utilities while in the process of working on other projects.

That backlog of commands has been gone through. For the next phase of Gen, we would like to build it out with a series of infrastructure improvements that lead up to making Gen an open community project that anyone can easily contribute to. Think of the code snippets sprinkled throughout wiki.tcl.tk, but with documentation and tests, and organized into a package (or packages).

If possible, we would like to first post up the designs and open it up for anyone to get involved, if they like.

To know when the date for the next release is determined, please follow the instructions below to subscribe to the mailing list.


Want to receive notifications about changes to Gen? You can subscribe to the announcements mailing list by sending an email to gen-announce-subscribe@robertbrogan.com. (No need for anything in subject or message body.) You will get an email every time we have a new release.

Want to see what other features that may be coming? Check out the wishlist.

Do you have a special request for something? You can use the feedback page and let us know. It never hurts to ask!
