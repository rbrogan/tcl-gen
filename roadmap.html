Roadmap
1.5
2015.05.04

This may will be the last release before doing a v2. We will try to finish up on miscellaneous commands, but may do several releases with both commands and infrastructure. When we move to v2, we will likely make breaking changes to the API all at once and then keep things the same for another cycle of releases until v3.

Quite possibly, there will be a several month interim between the last v1 release and the first release of v2.


Want to receive notifications about changes to Gen? You can subscribe to the announcements mailing list by sending an email to gen-announce@robertbrogan.com. Put subscribe for the subject. You will then get an email every time we have a new release.

Want to see what other features that may be coming? Check out the wishlist.

Do you have a special request for something? You can use the feedback page and let us know. It never hurts to ask!
