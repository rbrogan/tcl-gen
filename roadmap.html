Roadmap

Want to know what is coming up and when it will be out? You can check the following for upcoming release versions, dates and their features. (Please note this schedule is subject to change.)

1.1.0
2015.01.04

Commands for convenient SQLite interactions will come in this release.

TableExists

Check if a table can be found in the database.

CopyTable

Copy entries from one table to another.

LastId

Get the id of the last record.

WhereClauseFromDict

Convert a dict into a WHERE clause.

SetClauseFromDict

Convert a dict into a SET clause for use in an UPDATE.

RecordExists

Check whether a record matching the criteria exists.

InsertIfDoesNotExist

Insert a record if there is not already one.

InsertOrUpdate

Insert a record if none exists, but update it if it does.


1.2.0
2015.02.01

Datetime functionality will be featured in this release. Further commands TBD, watch for update.

Hhmmss2Seconds

Convert a string of the form HH:MM:SS to seconds.

Seconds2Hhmmss

Convert some number of seconds to a string of the form HH:MM:SS.

SumHhmmss

Sum up a list of time strings.

DiffHhmmss

Like integer subtraction, except it is for hh:mm:ss strings.

today

Get today's date.

yesterday

Get yesterday's date.

tomorrow

Get tomorrow's date.
