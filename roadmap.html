Roadmap

Want to know what is coming up and when it will be out? You can check the following for upcoming release versions, dates and their features. (Please note this schedule is subject to change.)

1.3
2015.03.02

The next release will feature Windows registry commands. We will also look to bring in some control and coroutine commands in this release. Here is an initial, tenative list of commands (subject to change):

RegKey2Dict

Read contents of registry key into a dict.

Dict2RegKey

Write contents of a dict under a registry key.

RegistryForEach

Iterate over registry contents in the manner of dict for.

RegistryWithEach

Iterate over registry contents in the manner of dict with.

RegistryWalker

A registry iterator in the form of a coroutine.

RegistryTrace

Put a trace on a registry value, similar to a Tcl variable trace.

RegistryPrint

Print the contents of a registry key or value.

LinkVarToRegValue

Set up a link such that changes to the Tcl variable cause changes to the registry value.

UnlinkVarFromRegValue

Stop propagating changes set up by LinkVarToRegValue.

IfOnFirst

Execute code if on first element of foreach loop.

IfOnLast

Execute code if on last element of foreach loop.


1.4
2015.04.02

This release is still being planned. At this point we are looking to bring in miscellaneous commands that did not make into the first four releases. We will also have as a stretch goal the initial implementation of a feature which allows you to generate / configure your own Gen source:

  1. Select which commands to include / exclude.
  2. Change names for commands and/or add aliases.
  3. Ability to put commands into namespaces and/or ensembles.


Want to see what other features that may be coming? Check out the wishlist.

Do you have a special request for something? You can use the feedback page and let us know. It never hurts to ask!
