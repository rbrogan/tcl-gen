Roadmap

Want to know what is coming up and when it will be out? You can check the following for upcoming release versions, dates and their features. (Please note this schedule is subject to change.)

1.2.0
2015.02.01

The next release will center on datetime convenience. Here is an initial, tenative list of commands (subject to change):

tod

Get the current time-of-day in hh:mm:ss format.

now

Get the current datetime.

today

Get today's date.

tomorrow

Get tomorrow's date.

yesterday

Get yesterday's date.

DatePlusDays

Return date n days after given date.

DateMinusDays

Return date n days before given date.

DayAfter

Return date just after given date.

DayBefore

Return date just before given date.

SumHhmmss

Add up time quantities of the form hh:mm:ss.

DiffHhmmss

Find difference between time quantities of the form hh:mm:ss.

Hhmmss2Seconds

Convert a time quantity of the form hh:mm:ss to an equivalent number of seconds.

Seconds2Hhmmss

Convert some number of seconds to a time quantity of the form hh:mm:ss.

IsDashedDateTime

Check whether the string is of the form YYYY-MM-DD hh:mm:ss.

IsDottedDateTime

Check whether the string is of the form YYYY.MM.DD.hhmm.

IsHhmmss

Check whether the string is of the form hh:mm:ss.

MultiplyHhmmss

Multiply a time quantity expressed in the form hh:mm:ss by some factor.

HhmmssIsAfter

Check whether the first time-of-day comes after the second.

HhmmssIsBefore

Check whether the first time-of-day comes before the second.

HhmmssIsBetween

Check whether the given time-of-day is between two others.

DateIsBefore

Check whether the first date comes before the second.

DateIsAfter

Check whether the first date comes after the second.

DateIsBetween

Check whether the given date is between two others.

DateTimeIsBefore

Check whether the first datetime comes before the second.

DateTimeIsAfter

Check whether the first datetime comes after the second.

DateTimeIsBetween

Check whether the given datetime is between two others.

DashedDateTime2DottedDateTime

Convert datetime of form YYYY-MM-DD hh:mm:ss to YYYY.MM.DD.hhmm.

DottedDateTime2DashedDateTime

Convert datetime of form YYYY.MM.DD.hhmm to YYYY-MM-DD hh:mm:ss.
