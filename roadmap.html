Roadmap

We hope to do a series of small releases (hopefully one every two weeks). Each release would consist of both a modest installment of commands as well as project improvements. Our main goal here from a project management POV is to keep things moving with commitments to hard dates and have those drive project improvements.

1.7
2015.08.24

In this release we will put out FTP commands for uploading, and possibly a few other assorted commands as well.

We will also try, if possible, to make it possible to load Gen without loading every package used. So, if you do not use FTP commands then the FTP package would not have to be loaded.


Want to receive notifications about changes to Gen? You can subscribe to the announcements mailing list by sending an email to gen-announce-subscribe@robertbrogan.com. (No need for anything in subject or message body.) You will get an email every time we have a new release.

Want to see what other features that may be coming? Check out the wishlist.

Do you have a special request for something? You can use the feedback page and let us know. It never hurts to ask!
