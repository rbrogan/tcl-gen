Roadmap

Want to know what is coming up and when it will be out? You can check the following for upcoming release versions, dates and their features. (Please note this schedule is subject to change.)

1.4
2015.04.02

Here are some of the miscellaneous commands you can expect to find (list is subject to change):

AddEpilogue

Add code to the end of a proc.

AddPrologue

Add code to the start of a proc.

AppendString2File

Open file, append string to end, close file.

SliceLeft

Like split, except it keeps the characters it splits on.

SliceRight

Like split, except it keeps the characters it splits on.

StringInsert

Insert value into target string, starting from a given index.

We will also have as a stretch goal the initial implementation of a feature which allows you to generate / configure your own Gen source:

  1. Select which commands to include / exclude.
  2. Change names for commands and/or add aliases.
  3. Ability to put commands into namespaces and/or ensembles.


1.5
2015.05.04

This likely will be the last release before doing a v2. We will try to finish up on miscellaneous commands. When we move to v2, we will likely make breaking changes to the API all at once and then keep things the same for another cycle of releases until v3.

Quite possibly, there will be a several month interim between this release and the first release of v2.

In the meantime, we may well be looking to enhance the functionality of the Gen source generator project.


Want to receive notifications about changes to Gen? You can subscribe to the announcements mailing list by sending an email to gen-announce@robertbrogan.com. Put subscribe for the subject. You will then get an email every time we have a new release.

Want to see what other features that may be coming? Check out the wishlist.

Do you have a special request for something? You can use the feedback page and let us know. It never hurts to ask!
